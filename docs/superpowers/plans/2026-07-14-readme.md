# 项目 README 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 创建一份兼顾项目访客与开发者的中文 `README.md`，准确介绍当前能力并提供可执行的构建运行说明。

**Architecture:** README 作为仓库入口，只保留项目摘要、技术架构和操作指南；详细玩法与实施决策通过相对链接指向现有 `docs/superpowers/` 文档。所有命令和路径均从当前工程配置中验证，不包含机器专属信息。

**Tech Stack:** Markdown、HarmonyOS 6.1.0(23)、DevEco Studio、Hvigor、HarmonyOS NDK C++、ArkTS、GLES3。

## Global Constraints

- README 使用中文，兼顾开源访客与本地开发者。
- 文档控制在约 150 至 250 行。
- 明确区分已实现能力、设计目标和已知限制。
- 不复制完整设计文档，不宣称未实现内容已经可玩。
- 不写入本机绝对路径、签名凭据或设备序列号。
- 兼容与目标 SDK 为 HarmonyOS 6.1.0(23)，API 23。
- Native 图形链路为 XComponent + NativeWindow + EGL + OpenGL ES 3。
